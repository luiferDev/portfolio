Voll Med est une application pour prendre des rendez-vous médicaux. J'ai utilisé Spring Boot pour le développement backend, authentification et autorisation via JWT token et Spring Security. Elle dispose d'un CRUD pour tous les endpoints et a été testée avec Insomnia.
