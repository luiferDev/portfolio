Areas Deportivas es un sistema para gestionar el alquiler de áreas deportivas. Implementé en el frontend un SPA con React Router v7, gestionando estados con Zustand. Usé ASP.NET Core en el backend, autenticación y autorización mediante token JWT y reservas de áreas deportivas.
