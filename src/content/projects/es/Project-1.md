Zorvanz es una aplicación de comercio electrónico para vender fragancias. Implementé en el frontend un SPA con React Router v7, gestionando estados con Zustand. Usé ASP.NET Core en el backend, autenticación y autorización mediante token JWT y gestión de pedidos y métodos de pago.
